# Court Data Fetcher & Judgment Downloader (MVP)

## Overview
This is a **minimum viable product (MVP)** for the internship assignment "Court Data Fetcher & Judgment Downloader".  
It allows a user to input Case Type, Case Number, and Year, and displays **dummy case details** with a PDF download button.  

**Note:** Real court portal scraping is not implemented yet; dummy data is used for demo purposes.

---

## Features
- Simple **UI** using Streamlit
- Input fields: Case Type, Case Number, Year
- Fetch button to display **dummy case details**
- Save queries in **SQLite database**
- **Download Judgment** button (dummy PDF)
- Handles empty input fields gracefully
